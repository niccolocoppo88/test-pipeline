# Canvas Drawing App

A simple HTML5 canvas drawing application with brush tools, color selection, and touch support.

## Live Demo

[![Live Demo](https://img.shields.io/badge/Live%20Demo-GitHub%20Pages-blue)](https://niccolocoppo88.github.io/test-pipeline/)

## Features

- Brush size control
- Color selection
- Eraser tool
- Clear canvas
- Download as PNG
- Touch support for mobile devices

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `app.js` - Application logic
