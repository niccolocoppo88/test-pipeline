# BRIEF — Canvas Drawing App (TEST PIPELINE)

**Progetto:** Web drawing app in HTML Canvas
**Tipo:** Prova / Proof of Concept
**Stack:** HTML5 + CSS3 + Vanilla JavaScript (no frameworks)

---

## 1. Concept & Vision

App di disegno minimal, immediata, elegante. L'utente apre la pagina e può disegnare subito — nessuna registrazione, nessuna complicatezza. L'estetica è quella di un tool professionale ma friendly: bianco, nero, un accento di colore.

Mood: **Faber Castell meets Linear** — precisione artigianale, interface pulita come un tool SaaS moderno.

---

## 2. Funzionalità Core

- **Drawing libero** con mouse e touch
- **Toolbar:** colori (palette predefinita), spessore pennello (slider), gomma
- **Azioni:** Clear canvas, Download PNG
- **Responsive:** Desktop + mobile

---

## 3. Design Direction

- **Palette:** bianco (#FFFFFF), nero (#1a1a1a), accento blu (#3B82F6), grigio (#6B7280)
- **Font:** Inter (Google Fonts)
- **Layout:** Toolbar fissa in alto, canvas full-screen sotto, footer minimale
- **UI:** Centrata, pulita, nessun bordo pesante
